# Line Multiplication Lab

A self-contained HTML/CSS/JS app demonstrating the Vedic/Japanese line multiplication method.

## Run

Open `index.html` in any modern browser. No build step and no network access are required.

## Features

- User-selected whole numbers up to four digits each
- Distinct colors for every digit-to-line group
- SVG animation for each line, diagonal grouping curve, intersection dot, dot count, carry, and final product
- Play, pause, reset, and single-step controls
- Speed control
- Carry table and diagonal dot-count panel
- Keyboard shortcuts: Enter in either number field to rebuild and animate, Space to play/pause, Right Arrow to step, R to reset

## Notes

Very large digits create many crossings, so the diagram can become visually dense. Smaller digits are clearer for instruction.
